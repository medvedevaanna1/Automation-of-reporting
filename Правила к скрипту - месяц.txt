1. Шаблон шаблон_заявки_по_жк.xsls (шаблон общий и для платных заявок, для заявок то и заявок по клинингу) должен находиться в той же папке
   что и сам скрипт (Ежемесячный.py). Шаблон содержит два листа: "платные_то", "клининг";

3. Шаблон должен отражать состав итоговых таблиц - первый стобец с перечнем ЖК, так же должны быть включены
   итоговые строки и столбец с названием управлении;

4. Выгрузки из 1С должны находиться в одной папке под названиями: 
   вк, гс, ритц - для всех видов заявок;   

5. Скрипт запускается из командной строки

   Что делать в командной строке:

   - Перейти в папку где лежит скрипт:
     
     cd /d полный_путь_к_папке_где_находится_скрипт

   - Вводим команду для запуска скрипта (скрипты запускаются по одному):

     python название_скрипта_с_расширением

   - Далее по программе скрипта появится текст: "Укажите полный путь к папке, в которой выгрузены файлы из 1С, затем нажмите Enter:"
     после : указываем полный адрес папки где лежат выгрузки из 1С и нажимаем Enter

   - После этого появится другой следующий текст: "Укажите полный путь к папке, где будет сохранена итоговвая таблица, затем нажмите Enter:"
     после : указываем полный адрес папки куда необходимо будет сохранить итоговую таблицу (excel-файл), нажимаем Enter

6. Далее скрипт полностью выполнится, создастся excel-файл (заявки_жк_месяц.xlsx с листами по видам заявок) в указаной папке, 
   в командной строке появится подный путь к папке в которой лежит скрипт;

   Скрипт выполнен.

   Далее перенести данные в отчет полным копированием таблиц с конечным форматированием.

Дополнительно:

Что необходимо для использования скрипта:

1. Установить python последней версии или 3.10
2. Через командную сроку загрузить библиотеки используемые в скрипте:
   
   pip install название_библиотеки

   библиотеки:

   - pandas
   - xlrd==1.2 (необходимо установить боллее позднюю версию 1.2 т.к. именно она сможет проситать excel-файл выгруженный из 1С)
   - numpy
   
   остальные библиотки используемые в скиптах стандартные и устанвливаются при установки python

- Посмотреть установленные библиотеки:

  pip freeze

- Посмотреть версию pip

  pip -V

- Обновить верию pip

  python -m pip install -–upgrade pip
